≪起動方法≫
「JJJ.exe」をダブルクリックして実行してください。

≪データの削除方法≫
セーブデータは以下の場所に生成されます。
Windows：%userprofile%\AppData\LocalLow\Micomprocedure\JJJ
Mac：~/Library/Application Support/MicomProcedure/JJJ

それぞれのファイルを削除することで初期状態からプレイできます。
オプション: Option.sav
ハイスコア: HighScore.sav
